HTML

HyperText Markup Language

The preferred markup language for Web pages is HTML. You can build your own website using HTML. Tim Berners-Lee, a scientist, created it in 1990. It facilitates communication between us and others on the World Wide Web (WWW).

CSS

Cascading Style Sheets

It works with how HTML components appear on the Web and affects how a website appears. It enables the separation of style from content. Additionally, it makes it simpler to exchange and maintain websites and style sheets.

Bootstrap

Botstrap Version 4

It is a free front-end framework that makes web development quicker and simpler. It enables you to make responsive designs quickly. It also adjusts how the web would look like to look good on any device.
